yet another translator
======================

Yet Another Translator using Google Translate API.

### Features

- Basic translation
- Any languages
- Dictionary
- Reverse translation
- **Colors!** (needs [colorama](https://pypi.python.org/pypi/colorama))

### Installation

- `pip install yat`, *or*
- `sudo python setup.py install`
